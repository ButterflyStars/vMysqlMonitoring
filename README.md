# vMysqlMonitoring

## OSX 10.12 (swift)

### Version

**1.0 Beta**

### 功能

- 实时修改数据库连接配置，并缓存到数据目录
- 程序启动，自动清空 general_log 内容
- 点击标题实现排序
- 实时筛选SQL语句
- 双击行复制SQL语句到剪贴板
